# API DeepSeek Bridge

Локальный проект-мост между Python и веб-версией DeepSeek через браузерное расширение.

## Кратко о проекте

- Python-клиент отправляет задачи в расширение по локальному WebSocket.
- Расширение выполняет задачу в DeepSeek (в браузере) и возвращает результат обратно.
- Поддерживаются очередь и параллельная обработка задач в расширении.
- Есть настраиваемый popup: адрес/порт подключения и селекторы элементов DeepSeek.

## Документация по частям
