Селекторы DeepSeek

CSS-селекторы элементов на странице чата (обновляй, если поменялась вёрстка).

Подсказка: как получить селектор
Вставляй CSS-селектор элемента, а не весь HTML.
1) DevTools → Elements → выбери элемент.
2) Copy → Copy selector и вставь сюда.
Если есть только outerHTML, найди стабильный class/атрибут и собери селектор вручную.
