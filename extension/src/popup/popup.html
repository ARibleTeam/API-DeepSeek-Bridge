Селекторы DeepSeek
Что вставлять в поля
Вставляй CSS-селектор элемента, а не весь HTML.
Как получить:
1) DevTools -> Elements -> выбери элемент.
2) Copy -> Copy selector и вставь сюда.
Если есть только outerHTML, найди по нему стабильный class/атрибут и сделай селектор.
